# PET empirical atlas report (2..100000)

## Scope

This report summarizes a bounded PET scan for integers in the closed range `2..100000`.

It is part of the PET empirical atlas track and is intended as a reproducible,
range-bounded structural snapshot rather than a statement of general theory.

## Reproduction

### Command

    python3 -m src.pet.cli scan 2 100000 --jsonl docs/reports/data/scan-2-100000.jsonl

### Summary command

    python3 tools/atlas_summary.py docs/reports/data/scan-2-100000.jsonl > docs/reports/data/atlas-summary-2-100000.txt

## Observations

1. The number of distinct PET shapes rises from the smaller `2..10000` baseline to `52` in `2..100000`, so the structural catalog is still expanding materially in this range.

2. The first observed PET of height `4` appears at `n = 65536`, and height `4` appears only once in this bounded scan.

3. The most common maximum branching values remain moderate (`2` and `3` dominate), while `max_branching = 6` is rare (`25` cases).

4. The first record reaching maximum branching is `30030`, while the first record reaching maximum node count is `25200`, again suggesting that width and total structural mass peak on different inputs.

## Interpretation and limits

This report is descriptive and range-bounded.

It does not claim asymptotic behavior, completeness of shape emergence, or permanence of the observed distributions beyond `2..100000`.

Its purpose is to provide a reproducible empirical atlas artifact for comparison with later scan ranges.
